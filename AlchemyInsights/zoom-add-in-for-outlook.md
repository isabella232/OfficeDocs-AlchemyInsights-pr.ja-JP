# <a name="zoom-add-in-for-outlook"></a>Outlook 用 Zoom アドイン

Zoom Scheduler アドインを使用すると、Web 上の任意の新規または既存の Outlook 予定表イベントに Zoom ミーティングを簡単に追加できます。

アドインは[ここから](https://go.microsoft.com/fwlink/?linkid=2126413)ダウンロードしてください。
