# <a name="edit-a-read-only-file"></a>読み取り専用ファイルを編集する

ドキュメント ライブラリで、ドキュメントの名前をクリックすると、予期せず読み取り専用ファイルとしてそのドキュメントが表示されることがあります。編集できるように開くには、[**開く**] (...) をクリックし、[**編集**] をクリックします。
