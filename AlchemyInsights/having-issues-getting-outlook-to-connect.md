# <a name="having-issues-getting-outlook-to-connect"></a>Outlook の接続に問題がある場合

次のドメインで Outlook が正常に動作するように構成されていないことを検出しました。
  
{HealthCheckString}
  
ドメイン (DNS) 設定の NS または CNAME レコードが適切に設定されていません。詳細については、「[任意の DNS ホスティング プロバイダーで DNS レコードを作成する](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/create-dns-records-at-any-dns-hosting-provider)」を参照してください。 
  
この問題を修正する場合は、Microsoft 365 管理センターの [[ドメイン]](https://admin.microsoft.com/adminportal/home#/Domains) ページに移動して支援を求めてください。
