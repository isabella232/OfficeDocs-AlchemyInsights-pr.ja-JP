# <a name="focused-inbox-in-outlook"></a>Outlook の優先受信トレイ

優先受信トレイでは、受信トレイが [優先] と [その他] という 2 つのタブに分類されます。最も重要なメッセージは [優先] タブにあり、残りのメッセージには引き続き簡単にアクセスできますが、[その他] タブに移動されます。
