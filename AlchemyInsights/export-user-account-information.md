# <a name="export-user-account-information"></a>ユーザー アカウント情報をエクスポートする

ライセンス、パスワード ポリシー、ブロックされたアカウント、AAD 同期状態などの情報を含む Microsoft 365 管理センターでユーザー アカウント情報をエクスポートするには、[**ユーザー**] > [[アクティブユーザー](https://go.microsoft.com/fwlink/p/?linkid=834822)] ページに移動し、[**ユーザーのエクスポート**] を選択します。
