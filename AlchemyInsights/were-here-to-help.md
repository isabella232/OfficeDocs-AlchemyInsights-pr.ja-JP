# <a name="were-here-to-help"></a>ご不明な点はお問い合わせください。

下記の **サポートにお問い合わせ** をクリックして、チケットを作成してください。 サポート担当者は、問題の解決に向けてすぐにご連絡します。

サブスクリプションが Microsoft パートナーによって管理されている場合は、パートナーと協力してサポートチケットを開きます。
