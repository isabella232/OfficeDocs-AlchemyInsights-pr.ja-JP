# <a name="what-are-contacts"></a>連絡先とは

Microsoft 365 では、さまざまな操作を連絡先に対して実行できます。その多くは連絡先の使い方によって決まります。

- Microsoft 365 全体のさまざまな連絡先の詳細については、「[クイック ヘルプ:連絡先を管理する方法](https://docs.microsoft.com/microsoft-365/admin/misc/ways-to-manage-contacts?view=o365-worldwide)」を参照してください。

- Outlook で連絡先を使い始めるには、「[Outlook で連絡先を使い始める](https://support.office.com/article/using-contacts-people-in-outlook-on-the-web-1e3438c7-26b2-420c-87de-3cea9d31b5cb?WT.mc_id=365AdminCSH&ui=en-US&rs=en-US&ad=US)」をご覧ください。
