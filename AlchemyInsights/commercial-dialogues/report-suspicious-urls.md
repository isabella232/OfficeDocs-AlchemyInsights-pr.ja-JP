# <a name="report-suspicious-urls"></a>不審な URL を報告する

不審な URL を送信するには、次の手順に従います。

1. **[新規の報告]** ポップアップに、プロトコル (https://) を含む完全な URL を入力します。
2. **[ブロックが必要だった]** を選択した場合は、URL が **[フィッシング]** または **[マルウェア]** のどちらであるかを指定します。
3. **[送信]** を選択します。
