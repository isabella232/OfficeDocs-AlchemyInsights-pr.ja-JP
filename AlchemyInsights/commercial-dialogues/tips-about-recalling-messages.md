# <a name="tips-about-recalling-messages"></a>メッセージを取り消すヒント

メッセージの取り消しは、以下のようにできる場合とできない場合があります。

* 管理者は、PowerShell を使用してユーザーの代理のメッセージを取り消すことができます。 管理センターからメッセージを取り消すことはできません。
* あなたはあなたの組織内の人々に送られたメッセージだけを取り消すことができます。 たとえば、Gmail アドレスに送信された場合は、メッセージを取り消すことはできません。
* **PC 版 Outlook 2016 から送信されたメッセージだけを取り消すことができます**。 ユーザーが、Outlook for Mac または Outlook on the web を使用してメッセージを送信する場合、取り消すことができません。
