# <a name="manage-app-groups-by-using-the-azure-portal-for-windows-virtual-desktop"></a>Windows Virtual Desktop 用の Azure portal を使用してアプリ グループを管理する

Windows Virtual Desktop の新しいホスト プール向けに作成される既定のアプリ グループには、完全なデスクトップも公開されています。 さらに、Azure portal を使用して、ホスト プールに 1 つ以上の RemoteApp アプリ グループを作成できます。

展開プロセストにより、以下の処理が実行されます。

1. RemoteApp アプリ グループを作成します。
2. 選択したアプリをアプリ グループに追加します。
3. 個々のユーザーまたはユーザー グループをアプリ グループに公開します。
4. 必要に応じて、アプリ グループを登録します。
5. 構成に応じて Azure Resource Manager テンプレートへのリンクを作成します。これにより、テンプレートをダウンロードして保存できます。

Windows Virtual Desktop 用の RemoteApp グループを作成するには、「[Azure portal を使用してアプリ グループを管理する](https://go.microsoft.com/fwlink/?linkid=2129550)」の手順に従います。
