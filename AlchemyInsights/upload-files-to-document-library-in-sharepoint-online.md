# <a name="upload-files-to-onedrive-or-sharepoint"></a>OneDrive または SharePoint にファイルをアップロードする

ファイルは SharePoint Online のドキュメント ライブラリに簡単にアップロードできます。アップロードするファイルをコンピューターからドラッグして、ドキュメント ライブラリにドロップするだけです。 OneDrive を利用すると、コンピューターとクラウドの間でファイルを同期することができ、どこからでもファイルにアクセスできるようになります。 また、同期されたファイルはエクスプローラーで直接操作して、オフラインであってもファイルにアクセスできます。 アップロードと同期の詳細については、次のリンクを参照してください。

- [ライブラリにフォルダーまたはファイルをアップロードする](https://support.office.com/article/upload-a-folder-or-files-to-a-document-library-eb18fcba-c953-4d45-8d90-8da66edeacdb)

- [同期を使用したアップロード](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-in-windows-615391c4-2bd3-4aae-a42a-858262e42a49)

- [ファイルの制限事項 (無効な文字、ファイルのアップロード サイズ、パスの長さなど)](https://support.office.com/article/invalid-file-names-and-file-types-in-onedrive-onedrive-for-business-and-sharepoint-64883a5d-228e-48f5-b3d2-eb39e07630fa)

- [OneDrive の同期に関する問題の解決方法](https://support.office.com/article/Fix-OneDrive-sync-problems-83ab0d8a-8400-45b0-8dcf-dc8aa8a6bcf8)

- [OneDrive Web サイトでのファイルのアップロードに関する問題を修復する](https://support.office.com/article/Fix-problems-uploading-files-on-the-OneDrive-website-9afcc4a0-e344-4bc9-9c9d-59d3e802247e)

- [アップロードとダウンロードの速度を最大化する](https://support.office.com/article/Maximize-upload-and-download-speed-8eeadfb8-501f-406d-997b-98ab6ff67f43)
