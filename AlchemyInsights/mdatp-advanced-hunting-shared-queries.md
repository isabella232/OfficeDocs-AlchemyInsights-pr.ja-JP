# <a name="sharing-advanced-hunting-queries"></a>高度な検索クエリの共有

高度な検索クエリは、同じ組織内のユーザー間で共有できます。 また、GitHub で公開されているクエリも検索できます。 これらのクエリを使用すると、クエリを最初から作成することなく、特定の脅威の捜索シナリオを迅速に実行できます。
  
詳細は、[高度な検索で共有クエリを使用する](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/advanced-hunting-shared-queries)をご参照ください。
